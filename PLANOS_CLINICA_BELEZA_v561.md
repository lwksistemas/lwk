## 🚀 COMO CRIAR UMA LOJA DE TESTE

### 1. Acesse o Superadmin
```
URL: https://lwksistemas.com.br/superadmin/login
```

### 2. Crie uma Nova Loja
- Clique em "Lojas" → "Adicionar Loja"
- Preencha os dados:
  - **Nome:** Clínica Teste Beleza
  - **Slug:** clinica-teste-beleza
  - **Tipo de Loja:** Clínica da Beleza
  - **Plano:** Escolha um dos 4 planos criados
  - **CPF/CNPJ:** (opcional)
  - **Endereço:** (opcional)

### 3. Crie um Usuário Admin
- Após criar a loja, crie um usuário administrador
- Email: admin@clinica.com
- Senha: (defina uma senha)

### 4. Acesse o Dashboard
```
URL: https://lwksistemas.com.br/loja/clinica-teste-beleza/dashboard
```

### 5. Popule Dados de Teste (Opcional)
```bash
cd backend
source venv/bin/activate
python criar_dados_clinica_beleza.py
```
