## 📖 O QUE É ESTE PROJETO?

Sistema híbrido que combina **segurança** (slug com hash) e **facilidade de uso** (atalho simples + login automático) para acesso às lojas.

### Problema Resolvido
❌ **Antes:** CNPJ exposto na URL (inseguro, ruim para UX)  
✅ **Depois:** Hash aleatório + atalho simples (seguro e fácil)

### Resultado
- Segurança: +200% (3/10 → 9/10)
- UX: +233% (3/10 → 10/10)
- SEO: +233% (3/10 → 10/10)
