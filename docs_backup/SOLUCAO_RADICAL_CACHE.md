- Cache de memória RAM

### Solução 4: Modo Desenvolvedor (Avançado)

**No Chrome/Edge:**
1. Pressione F12
2. Vá em **Application** > **Storage**
3. Clique em **"Clear site data"**
4. Marque TODAS as opções
5. Clique em **"Clear data"**
6. Vá em **Network**
7. Marque **"Disable cache"**
8. Mantenha o DevTools aberto
9. Recarregue a página (Ctrl+Shift+R)
10. Faça login

### Solução 5: Limpar Cache do Sistema Operacional

**Linux:**
```bash
# Limpar cache do Chrome
rm -rf ~/.cache/google-chrome/
rm -rf ~/.config/google-chrome/

# Limpar cache do Firefox
rm -rf ~/.cache/mozilla/
rm -rf ~/.mozilla/firefox/*/cache2/
```

**Windows:**
```
1. Pressione Win+R
2. Digite: %LocalAppData%\Google\Chrome\User Data
3. Delete a pasta "Cache"
4. Delete a pasta "Code Cache"
```

**Mac:**
```bash
# Limpar cache do Chrome
rm -rf ~/Library/Caches/Google/Chrome/
rm -rf ~/Library/Application\ Support/Google/Chrome/

# Limpar cache do Safari
rm -rf ~/Library/Caches/com.apple.Safari/
```
