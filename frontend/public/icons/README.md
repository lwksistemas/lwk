# Ícones PWA

- **icon.svg** – ícone em SVG (já incluso; tema rosa #ec4899).
- Para melhor suporte (Android/iOS), adicione:
  - **icon-192.png** – 192×192 px, fundo sólido (ex.: rosa #ec4899).
  - **icon-512.png** – 512×512 px, mesmo desenho.

Ícones quadrados, fundo sólido, ficam melhores na tela inicial.
